    Introduction

    Welcome to the EventsManager API! This API was built with NodeJs for Users of the Events Manager App to be able to Create Events, Modify their Events, Delete their Events and Aministrators should be able to create event centers, Delete Center and Modify Centers

    Authentication

    Authentication has not been implemented

    EventsManager Centers and Events API

    Get All Centers

    GET http://localhost:4000/api/v1/centers

    The above command returns JSON structured like this:

    [
      {
        id: 1,
        name: 'Python Hall',
        location: 'Lagos',
        facilities: ['Air conditioners', 'Projectors']
      },
      {
        id: 2,
        name: 'Python Hall',
        location: 'Lagos',
        facilities: ['Air conditioners', 'Generators']
      },
      {
        id: 3,
        name: 'Python Hall',
        location: 'Lagos',
        facilities: ['Air conditioners', 'Projectors', 'Generators']
      },
      {
        id: 4,
        name: 'Python Hall',
        location: 'Lagos',
        facilities: ['Projectors', 'Generators']
      }
    ];

    This endpoint retrieves all Centers

    HTTP Request

    GET http://localhost:4000/api/v1/centers


    RETURNS STATUS CODE 200

    Get a Specific Center

    http://localhost:4000/api/v1/centers/<centerId>

    The above command returns JSON structured like this:

    {
        id: 4,
        name: 'Python Hall',
        location: 'Lagos',
        facilities: ['Projectors', 'Generators']
      }

    This endpoint retrieves a specific Center

    Edit a Specific Center

    PUT http://localhost:4000/api/v1/centers/<centerId>

    The above command returns JSON structured like this:

    {
        id: 4,
        name: 'Python Hall',
        location: 'Lagos',
        facilities: ['Projectors', 'Generators']
      }

    This endpoint Edits a specific Center

    It will return errors for a center of that ID does not exist.

    DELETE http://localhost:4000/api/v1/centers/<centerId>

    Edit a Specific Center

    DELETE http://localhost:4000/api/v1/centers/<centerId>

    The above command returns JSON structured like this:

    {
        message: 'success',
        error: false
      }

    This endpoint Deletes a specific Center

    It will return errors for a center of that ID does not exist.

    DELETE http://localhost:4000/api/v1/centers/<centerId>

    Errors

    The EVENTSMANAGER API uses the following error codes:

    Error Code Meaning
    200 OK
    400 Bad Request – Your request sucks
    404 Not Found – The specified kitten could not be found
    500 Internal Server Error – We had a problem with our server. Try again later.
